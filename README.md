# Dictionary
A simple dictionary application implemented in F#.

## Features

+ Add/update/delete words
+ Case-insensitive search
+ Partial match search 
+ Save/load data into JSON file
+ Error Handling

## Getting Started
To run this project, you must have the following installed on your system:

+ .NET SDK (version 6.0 or later)
+ Visual Studio, VS Code, or another F#-capable editor

1. Clone the repository
```bash
git clone https://github.com/nadaabdulghaffar/Dictionary.git

```
